# Complaint-Management-System
This is a project to handle complaints at college level implemented using pyhton and files. User can be of 3 categories 1.)Studemt 2.)CR 3.)Principal
Student can register a complaint about certain event happened.
Complaints are recorded in a centralized database for easy tracking.
CRs and principals can access the system to solve, forward, and resolve complaints.
Communication and collaboration among stakeholders are facilitated through remarks and solutions.
The system ensures timely resolution of complaints and maintains transparency.
